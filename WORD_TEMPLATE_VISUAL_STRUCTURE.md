## MSP Tone Writing Pattern

```
┌─────────────────────────────────────────────────────────────────┐
│                    MSP Monthly Update Tone                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Opening Pattern:                                               │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ "This month, we [action verb]..."                         │ │
│  │ "Our team observed..."                                     │ │
│  │ "During our review of your environment..."                │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  Perspective:                                                   │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ Use "we" for ICB Solutions                                │ │
│  │ Use "you/your" for the client                             │ │
│  │ Use "your organization" for formal reference              │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  Content Focus:                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ ✓ Business impact                                         │ │
│  │ ✓ Risk mitigation                                         │ │
│  │ ✓ Cost optimization                                       │ │
│  │ ✓ Operational efficiency                                  │ │
│  │ ✗ Technical jargon without explanation                   │ │
│  │ ✗ Passive voice                                           │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  Closing Pattern:                                               │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ "We recommend..."                                          │ │
│  │ "Our team will focus on..."                               │ │
│  │ "We're committed to..."                                    │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## Color Coding Reference

```
Priority Levels:
├─ 🔴 High Priority    → Red (#ef4444)    → Urgent, critical impact
├─ 🟠 Medium Priority  → Orange (#f59e0b) → Important, moderate impact
└─ 🟢 Low Priority     → Green (#10b981)  → Nice to have, low impact

Section Categories:
├─ 🔐 Identity Status  → Files starting with "I"
├─ 🛡️ Security Status  → Files starting with "S"
└─ 💻 Endpoint Status  → Files starting with "E"
```

## Data Flow Through System

```
┌──────────────────────────────────────────────────────────────────┐
│                    Screenshot Capture                            │
│  I_licenses.png, S_summary.png, E_devices.png, etc.            │
└──────────────────────────────────────────────────────────────────┘
                            ↓
┌──────────────────────────────────────────────────────────────────┐
│              OpenAI Analysis (GPT-4o Vision)                     │
│  • Analyzes each screenshot                                      │
│  • Generates 150-200 word summary in MSP tone                    │
│  • Creates 3-5 prioritized recommendations with rationale        │
│  • Assigns priority levels and time estimates                    │
└──────────────────────────────────────────────────────────────────┘
                            ↓
┌──────────────────────────────────────────────────────────────────┐
│            Word Document Generator                               │
│  • Categorizes screenshots by first letter (I/S/E)               │
│  • Creates 4-section structure                                   │
│  • Centers images without titles                                 │
│  • Adds Comments sections with paragraph analysis                │
│  • Builds Priority Areas subsections (3-5 per section)           │
│  • Generates Call to Actions with top 9 priorities               │
└──────────────────────────────────────────────────────────────────┘
                            ↓
┌──────────────────────────────────────────────────────────────────┐
│                  Generated Report                                │
│  CustomerName_Health_Report_October_2025.docx                   │
└──────────────────────────────────────────────────────────────────┘
```
